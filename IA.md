Registro de uso de Inteligência Artificial - AgroOrbit IA

Este arquivo documenta o uso de IA no complemento da disciplina de Web Development, conforme solicitado na atividade.

Interação 1
O que foi solicitado para a IA

Auxílio na criação de algumas funcionalidades em JavaScript para complementar o projeto AgroOrbit IA, como validação de formulário, quiz interativo e pequenos recursos visuais.

O que a IA retornou

Sugestões de código em JavaScript puro para implementar as funcionalidades solicitadas e integrar os recursos ao projeto já desenvolvido pelo grupo.

O que foi alterado ou rejeitado e motivo

Foram realizados ajustes manuais pelos integrantes para adaptar o código à estrutura existente do projeto e manter a identidade visual definida pela equipe.
## Interação 2

### O que foi solicitado para a IA

Orientação sobre a organização dos arquivos do projeto, separando corretamente HTML, CSS e JavaScript.

### O que a IA retornou

Sugestões para distribuir o conteúdo entre os arquivos index.html, css/style.css, css/reset.css e js/script.js, seguindo boas práticas de desenvolvimento web.

### O que foi alterado ou rejeitado e motivo

Algumas sugestões foram adaptadas para atender às necessidades específicas do projeto e manter a estrutura já utilizada pelo grupo.
